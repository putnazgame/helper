# ระบบช่วยเหลือผู้พิการครึ่งซีก

## การติดตั้ง

### 1. ติดตั้ง Python (3.10+)
ดาวน์โหลดจาก https://python.org

### 2. ติดตั้งไลบรารีที่จำเป็น
```
pip install pygame pyttsx3
```

### 3. (ตัวเลือก) ติดตั้ง gTTS สำหรับเสียงภาษาไทยที่ดีกว่า
```
pip install gtts
```
> gTTS ต้องการอินเทอร์เน็ต แต่เสียงภาษาไทยชัดกว่า

### 4. เปิดโปรแกรม
```
python main.py
```

## การควบคุม (มือซ้าย)

| การกระทำ | คีย์บอร์ด | Joystick |
|----------|-----------|----------|
| เลื่อนขึ้น | ↑ | L-Stick ↑ หรือ D-pad ↑ |
| เลื่อนลง | ↓ | L-Stick ↓ หรือ D-pad ↓ |
| ตกลง/พูด | Enter / Space | L1 หรือ L2 หรือกดก้านซ้าย |
| ออก | Escape | — |

## ปัญหาที่พบบ่อย

**เสียงออกมาเป็นภาษาอังกฤษ**
→ ไปที่ Settings > Time & Language > Speech แล้วติดตั้ง Thai language pack
→ หรือติดตั้ง gTTS แทน (`pip install gtts`)

**ภาษาไทยแสดงเป็นสี่เหลี่ยม (□)**
→ ตรวจสอบว่ามีฟอนต์ Tahoma หรือ Leelawadee ในระบบ (Windows มักมีอยู่แล้ว)

**จอยไม่ตอบสนอง**
→ เสียบจอยก่อนเปิดโปรแกรม
→ ตรวจสอบ CONFIRM_BUTTONS และ LANALOG_Y_AXIS ในโค้ด (บรรทัด ~50)
